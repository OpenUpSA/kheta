# Layout

![stability-stable](https://img.shields.io/badge/stability-stable-green.svg)

**This is a component that should wrap all views.**

It does the following:
- Adds header and/or footer (unless disabled)
- Forces vertical scrollbar, so that content does not jump if scrollbar appears.
